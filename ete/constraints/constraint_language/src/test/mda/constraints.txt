

Définition
    un pilote est certifié pour un avion :
        le modèle des certificats du pilote contient le modèle de l'avion !

Définition
    un pilote est habilité pour un avion :
        le pilote est certifié pour le modèle de l'avion !

Règle le commandant est certifié :
        le commandant du vol est habilité pour l'avion du vol.
Règle le copilote est certifié :
        le copilote du vol est habilité pour l'avion du vol.
